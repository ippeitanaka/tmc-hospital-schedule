# CSVテンプレート利用ガイド

このフォルダには、病院実習スケジュール管理システムにデータをインポートするためのCSVテンプレートが含まれています。

## ファイル構成

### 1. csv-template-students.csv (学生情報)

学生の基本情報を管理するファイルです。

**カラム構成:**
- `学籍番号`: 7桁の数字（例: 2420001）
- `氏名`: 学生の氏名（例: 山田 太郎）
- `ふりがな`: カタカナ表記（例: ﾔﾏﾀﾞ ﾀﾛｳ）
- `性別`: 男 または 女
- `生年月日`: YYYY/M/D形式（例: 2005/4/1）
- `年齢`: 数字（例: 20）
- `実習施設名`: 実習先の病院名（例: 大阪市立総合医療センター）
- `昼夜`: A または B
- `班名`: 班の名称（例: A班1）

### 2. csv-template-schedules.csv (スケジュール情報)

学生の日程情報を管理するファイルです。

**カラム構成:**
- `学籍番号`: 学生情報と紐付ける7桁の数字
- `日付`: YYYY-MM-DD形式（例: 2026-01-20）
- `記号`: スケジュールの種類を示す記号
  - `〇`: 病院実習当日
  - `明`: 実習明け（〇の翌日は自動で明になります）
  - `学`: 学校登校日
  - `数`: 数学セミナー
  - `半`: 午後のみ登校
  - `オリ`: オリエンテーション
- `備考`: 任意のメモ（空欄可）

## 使用方法

1. テンプレートファイルをダウンロードまたはコピー
2. Excelやスプレッドシートで開く
3. 実際のデータを入力（ヘッダー行は削除しない）
4. CSV形式で保存（UTF-8エンコーディング推奨）
5. アプリケーションの「データをインポート」機能でアップロード

## 注意事項

- **文字コード**: UTF-8で保存してください
- **学籍番号**: students.csvとschedules.csvで一致する必要があります
- **日付形式**: schedules.csvは必ずYYYY-MM-DD形式で入力してください
- **記号**: 定義された記号以外は使用しないでください
- **カンマ**: データ内にカンマが含まれる場合は、ダブルクォーテーションで囲んでください

## サンプルデータ

テンプレートにはサンプルデータが含まれています。実際のデータに置き換えてご使用ください。

## トラブルシューティング

### インポートエラーが発生する場合
- ファイルがUTF-8で保存されているか確認
- 学籍番号が7桁の数字になっているか確認
- 日付形式が正しいか確認（YYYY-MM-DD）
- 必須項目が空欄になっていないか確認

### データが正しく表示されない場合
- schedules.csvの学籍番号がstudents.csvに存在するか確認
- 記号が定義されたもの（〇、明、学、数、半、オリ）になっているか確認
